# dotfiles only related to cli

## CLI LIST

### install & configuration

- **git**

  - `~/.gitconfig`

  - `~/.gitignore`

- **zsh**

  - **oh-my-zsh** 설치 

  - **lambda** 프롬프트 설치 

  - **zsh-autosuggestions** 플러그인 설치 

  - `~/.zsh_aliases`

- **vim**

  - **monokai-pro** 컬러 테마 설치 

  - **junegunn-vimplug** 설치 

  - `~/.vimrc`

- **tmux**

  - `~/.tmux.conf`

- **gdb**

  - **pwndbg** 커스터마이징된 설치 

  - `~/.gdbinit`

### only install

- **unzip**

- **fd**

- **bat**

- **exa**

## usage

```shell
./install.sh
```

- supportted os

  - Arch Linux

  - Ubuntu Linux
