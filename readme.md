# dotfiles only related to cli

## usage

```shell
./install.sh
./install_pwngdb.sh # if you need gdb
```

- supportted os

  - Arch Linux

  - Ubuntu Linux

  - Kali Linux

## CLI list

- **git**

  - `~/.gitconfig`

  - `~/.gitignore`

- **zsh**

  - **oh-my-zsh**

  - **alien-minimal** 

  - **zsh-autosuggestions**

  - `~/.zsh_aliases`

- **vim**

  - **onedark** 

  - **junegunn-vimplug**

  - `~/.vimrc`

- **tmux**

  - `~/.tmux.conf`

- **gdb**

  - **pwndbg** 

  - `~/.gdbinit`

- **unzip**

- **fd**

- **bat**

- **exa**
